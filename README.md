# NoteVault

NoteValut is responsive note-taking application built with **React** and **Vite**. It provides a clean and intuitive interface for creating, organizing, and managing your notes efficiently.

### Features

* Create and save notes instantly.
* Pin important notes so they always stay at the top.
* Move unwanted notes to the **Bin** instead of permanently deleting them.
* Automatic data persistence using **Local Storage**, so your notes remain available even after refreshing or reopening the browser.
* Clean, responsive, and user-friendly interface.

### Tech Stack

* React
* Vite
* JavaScript
* TailwindCSS
* Local Storage API

### Project Goal

This project was built to practice modern React development, state management with hooks, component-based architecture, and responsive UI design while creating a practical application that can be used daily.
